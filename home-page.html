Project Overview

The project is a fan-made one of the original game made by N called "Schoolgirl Supervisor - Saori Sato". Later additions of the series exist with "Schoolgirl Supervisor - Wildlife" and "JP Schoolgirl Supervisor Multi(player)". As such, the project has been named "JP4S", which stands for "Japanese Schoolgirl Supervisor - Saori Sato".
Project Technologies

The game is being developed in the Unity 3D Engine with C#. It currently employs the xNode plugin for the dialogue system. As for the 3D models, a combination of VRoid Studio and Blender is used. Currently, no sound system has been implemented. The game will not feature online-functionality, meaning that it can be played even with no connection.
Target Experience

The target audience is people 16 years or older. The game is NOT meant for children as it covers potentially heavy topics. There will be no nudity but there may be some gore content/content about death.
The user experience needs to be carefully considered when designing this game. As such, during its development, you should be able to see the following:
  • Compatibility
    • The products should be available for many PCs. This means that the performance should be carefully monitored and the game should be compiled for both Windows and GNU/Linux-based operating systems.
  • Accessibility
    • The products should be usable by many people, regardless of their capabilities. Things like visual impairment or auditory issues are going to be respected and considered.
  • Security
    • The products should not interfere with the safety of anyones devices and applications need to be tested properly.
  • Cohesion
    • The products will have a defining theme to aid in recognision.
  • Creativity
    • The game is free to be moddified under the condition that it is as freely available as the original game and does not cause any harm to anyones devices. (See the license on the official GitHub repository.)
(Please note, that products include both the game and the website in most cases.)
